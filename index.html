ComplyMap API

Нормализованные данные реестров сертификации и деклараций РФ через JSON API. Обновление данных ≤ 2 часов. Соответствие 152-ФЗ.

Преимущества

Стабильный API

Один endpoint для поиска по ИНН, номеру декларации, ТН ВЭД. Курсорная пагинация для больших объёмов.

Нормализованные данные

Все наименования, адреса и коды приведены к единому стандарту. Никакого мусора из первоисточника.

Актуальность ≤ 2 часов

Синхронизация с официальными реестрами не реже 1 раза в 2 часа. Данные всегда свежие.

Соответствие 152-ФЗ

Агрегируем только данные из открытых реестров (ЕГРЮЛ/ЕГРИП, Росаккредитация). Юридическая чистота.

Для кого

Для Агрегаторов

B2B-справочники, тендерные площадки, сервисы проверки контрагентов. Масштабная проверка поставщиков без риска блокировок.
